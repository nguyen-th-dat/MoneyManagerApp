# Money Manager App (Ứng dụng Quản Lý Chi Tiêu)

Ứng dụng **Quản lý chi tiêu cá nhân trên Android** giúp người dùng theo dõi thu nhập, chi tiêu, phân loại giao dịch và thống kê tài chính theo thời gian. Mục tiêu của app là hỗ trợ quản lý tài chính cá nhân hiệu quả và trực quan.

---

## Công nghệ sử dụng

* **Ngôn ngữ:** Java
* **IDE:** Android Studio
